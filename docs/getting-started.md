# Getting Started

## Installation

Install using a package manager like [pnpm](https://pnpm.io), [yarn](https://yarnpkg.com/), or [npm](https://www.npmjs.com).
  
::: code-group
```bash [pnpm]
pnpm i vue-interface
```

```bash [yarn]
yarn add vue-interface
```

```bash [npm]
npm i vue-interface
```
:::
